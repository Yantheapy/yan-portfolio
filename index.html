Investment analytics · web application

Board-grade financial decision tool

UAE multi-year medical investment project — as Director of Special Projects, QUBE Development
Challenge

The board needed a defensible go/no-go on a multi-year clinic investment — not a static spreadsheet only one person could read.

What I built

A live parametric model turned into an interactive web app: a six-driver scenario engine (Base / Upside / Downside), live NPV / IRR / MOIC / payback, Monte-Carlo simulation, DCF waterfall, tornado and two-way sensitivity, investor-vs-founder return models, Excel ingestion, EN/RU and PDF export — productizing a multi-thousand-formula model.

Impact

A spreadsheet became a board-grade decision tool: any scenario answered in seconds, assumptions transparent, capital allocation defensible.

Operating system · full platform

Clinic operations & governance platform

Premium DHA-regulated dental clinic, Dubai — as CEO
Challenge

Scaling a regulated clinic meant governance, quality, sales and operations lived in scattered documents — no single, access-controlled source of truth.

What I built

A 190+ page internal platform on a custom engine: sales-enablement system, 29 SOPs, CAPA / quality, patient-risk and RAID registers, complaint-case workflows — fully bilingual RU/EN.

Engineering

Edge-middleware auth (HMAC-signed token cookies) plus email one-time-code login for clinicians; 14 serverless API endpoints; a key-value data layer with optimistic locking, audit trail and automatic backups; usage telemetry; CI/CD from GitHub to Cloudflare.
